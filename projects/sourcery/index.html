SOURCERY

A Web Appliction
designed to help bootcamp students thrive.

The challenge

Sourcery began as my final group project of my coding bootcamp. We were searching for a way to share, save, and filter through vast amounts of coding resources. Our project was verion 1.0, called Resource Roundup. After graduating I decided to continue developing this project and created version 2.0, SOURCERY. I learned to use open-source code, an was able to build and extensive application that not only served as a resource library, but a Github powered bootcamp project hub.

Launch ProjectGithub Repo

Creative
rebranding

My first move was to create a "Cohort Code" sign-up feature. This kept the app to only showing and sharing information between bootcamp peers.
I built a library of React Components to create consistancy and flow throughout the app. I resused cards for both project and resource features. I introduced tags, filtered searching, and a cohesive user dashboard.
The project reached new levels with the implimentation of Github OAuth. Now the app dynamically pulled relavent data, plus made an activity feed and photo personalization possible..

The features were designed to serve students
both during their bootcamp and post graduation.

Users can create unlimited resources, then filter through their cohorts library and save or like whatever meets their needs. Users also create projects, and set up teams. Projects have specialty member blocks, so teammates have private chats and administrative features in their view. Public, non team members can still view all project ideas, request to join, and share their thoughts and views.

Extra attention to detail development

Our team struggled with v1.0. We weren't able to deploy the site, and didn't build any features beyond our MVP. In developing v2.0, I grew grit and skills as a developer. From design and mockups, to never-ending back-end code; I am incredibly proud of my work.

Every element
made with love

The app lets students collaborate in a new, organized way.
Dynamic data is shown is a useful and comprehendable way.
Group projects have never been so easy to manage, explore or join.
From colors to tone styles to background effects, everything is meant to inspire students to remember the magic of building coding skills.
